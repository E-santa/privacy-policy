PRIVACY POLICY

  1. ADVERTISEMENTS

    No advertisements are displayed to any users.

  2. DATA COLLECTION

    We collect no sensitive user data. All calculation and analysis is local on the device running the application. No user data is even relayed back to the backend because there is no backend. It is a self-contained app.

  3. SOCIAL

    There is no social aspect to this application and no user data is shared with any other users.

  4. APIs AND SDKs

    No APIs or SDKs that collect any user data or otherwise have an age screen, are used in this application.

  5. CONTENT

    All content is appropriate for all ages.
