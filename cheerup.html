PRIVACY POLICY

  1. ADVERTISEMENTS

    No advertisements are displayed to any users.

  2. DATA COLLECTION

    We collect zero user data whatsoever. All calculations and analyses are done locally on the device running the application. No user data is even relayed back to the backend because there is no backend. It is a self-contained app.

    User faces are accessed through the camera in real time only while the app is being used, and this data is not stored anywhere. No other user data is accessed by the app.

    No user data is shared with anyone.

  3. SOCIAL

    There is no social aspect to this application and no user data is shared with any other users.

  4. APIs AND SDKs

    No APIs or SDKs that collect any user data or otherwise have an age screen, are used in this application.

  5. CONTENT

    All content is appropriate for all ages.
